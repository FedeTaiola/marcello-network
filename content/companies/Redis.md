# Redis

## Cos'è
Redis (REmote DIctionary Server) è un database in-memory open source fondato da Salvatore Sanfilippo (antirez). È noto per la sua velocità estrema, utilizzato come database, cache, message broker e queue dalle infrastrutture tech più grandi del mondo (es. Twitter, GitHub, Snapchat). L'azienda dietro il progetto commerciale si chiama Redis Ltd. (in precedenza Redis Labs).
